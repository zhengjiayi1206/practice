# Using Double Deep Q-Network to Learn How To Play Flappy Bird based on Deep Q-Network (https://github.com/yenchenlin/DeepLearningFlappyBird)

## Installation Dependencies:
* python == 3.10
* torch == 2.2.1
* numpy == 1.26.4
* opencv-python == 4.9.0
* pygame == 2.5.2

## How to Run?
```
git clone https://github.com/zizzey2002/DeepLearningFlappyBird
cd DeepLearningFlappyBird
python torch_gpt_DDQL.py
```
